# AI for Science 市场与岗位地图

> 面向想进入 AI for Science / AI4Science 方向的同学和研究者，整理国内外值得关注的公司、实验室和研究机构。覆盖生物医药、化学、材料、能源、物理模拟、气候与科研基础设施等方向。

更新时间：2026-06-14  
说明：本清单按公开信息整理，优先收录“有代表性技术、产品、论文、平台或产业落地”的组织；招聘状态请以官网为准。

## 市场概览

AI for Science 正从“模型论文”走向“闭环研发系统”。最活跃的商业场景集中在五类：

| 方向 | 典型问题 | 市场成熟度 | 主要岗位 |
|:---|:---|:---|:---|
| AI 制药与蛋白设计 | 靶点发现、小分子生成、抗体/蛋白设计、ADMET | 高 | 算法、计算化学、结构生物、药化、平台工程 |
| 材料与能源 | 电池、催化、半导体、合金、高分子、固态材料 | 中高 | 材料计算、分子模拟、机器学习势能、实验自动化 |
| 科学计算与仿真 | DFT、分子动力学、PDE、气候、流体、等离子体 | 中 | 科学机器学习、高性能计算、数值算法、仿真平台 |
| 自动化实验室 | 机器人实验、高通量筛选、主动学习闭环 | 中 | 机器人、实验平台、数据工程、实验科学家 |
| 科研基础设施 | 科学数据、知识库、智能体、云计算平台 | 早中期 | 后端/平台、数据治理、科学智能体、MLOps |
| 药企与 CRO 数字化研发 | CADD、ADMET、临床数据、工艺优化、药物安全 | 中高 | 计算化学、生物信息、临床数据科学、研发平台 |
| 算力与芯片基础设施 | 量子计算、超算、GPU/NPU、云上科学计算 | 中 | HPC、算子优化、芯片编译器、分布式训练 |

**核心判断**

- 国内机会更集中在制药、材料、能源、科研云平台和产业级实验室自动化。
- 国外头部玩家更强调基础模型、蛋白/药物管线、生成式材料、科研智能体和大型科研基础设施。
- 初创公司更看重“算法 + 领域知识 + 可验证实验结果”，大厂实验室更看重基础模型、平台化和跨学科协作。
- 大型药企、CRO/CDMO、新能源、半导体和算力公司也在吸纳 AI4S 人才，岗位名称常表现为计算化学、仿真算法、HPC、数据科学或研发数字化。
- 研究所和高校仍是新模型、新数据集、新范式的重要来源，也是进入产业界前积累论文和领域理解的关键入口。
